Digitalspeeed is a tech blog covering topics like cloud solutions, software architecture,etc.

I deployed a WordPress website on AWS EC2, designed using Elementor. It's my personal blog where I write about my tech projects, adventures, and new skills I've learned. Initially, I set up the website on AWS Lightsail, but I later moved it to EC2 to explore more cost-effective solutions.

I also integrated backend logic to process data from newsletter form submissions, sending it directly to an AWS Lambda function, which then stores the information in a DynamoDB database.>
